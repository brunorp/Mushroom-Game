## Installation and Usage
in cmd, choose a folder and run the command:
 
 ```bash
    git clone https://github.com/brunorp/Mushroom-Game.git
 ``` 
Go to the "Mushroom-Game" folder and run:

```bash
    npm install
``` 

on cmd, run:

```bash
   node service/service.js
```
